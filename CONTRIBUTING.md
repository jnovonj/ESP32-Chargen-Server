# Contributing to ESP32-Chargen-Server

Thank you for your interest in contributing! This guide provides a quick overview of how to report issues, propose changes, and submit pull requests.

## Code of Conduct
This project and everyone participating in it is governed by the `CODE_OF_CONDUCT.md`. By participating, you are expected to uphold this code.

## How to Report an Issue
- Search existing issues first to see if your problem has already been reported.
- If not, open a new issue and describe:
  - What happened (the observed behavior).
  - What you expected to happen.
  - Minimal steps to reproduce the issue (if applicable).
  - Your environment (hardware, PlatformIO version, framework version).
  - Any relevant logs or error messages.

## How to Propose Changes (Pull Request)
1. Fork the repository and clone your fork locally.
2. Create a descriptive branch: `git checkout -b feat/my-new-feature` or `fix/my-bug-fix`.
3. Make your changes on the new branch.
4. Ensure the code compiles and does not break existing functionality.
5. Add tests or a clear description of the manual testing you performed (if applicable).
6. Commit your changes with clear, atomic messages. Examples:
   - `feat: Add support for X`
   - `fix: Correct race condition in Y`
   - `docs: Update README regarding Wi-Fi setup`
7. Push your branch to your fork and open a pull request to the `main` branch of this repository.
8. In the PR, briefly describe the change, the reason for it, and any relevant impact.

## Style and Best Practices
- Keep the code readable with descriptive names.
- Avoid massive formatting changes in commits that don't add functionality.
- Prefer small, self-contained commits.

## License and Third-Party Files
- This project is licensed under GNU GPL v3 (see `LICENSE.md`). Ensure any contribution you submit is compatible with GPLv3.
- If you include third-party code, please state its source and license.

## Questions and Help
If you need help getting the project running or making a change, feel free to open an issue, and we can look at it together.
